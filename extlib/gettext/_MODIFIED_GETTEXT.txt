This copy of GNU gettext-runtime 1.0 is a modified version of the original.

Release version: gettext-1.0.tar.gz

The following changes have been made to the original:

- Removed most autoconf infrastructure.

- Removed all libraries except gettext-runtime/intl/.

- Added a CMakeLists.txt file for CMake integration.

To obtain the original GNU gettext-1.0, visit:
https://www.gnu.org/software/gettext/

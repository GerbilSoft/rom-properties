This copy of libpng-1.6.40 is a modified version of the original.

The following changes have been made to the original:

- autoconf build infratsructure has been removed.

- Files not necessary for building libpng have been removed.

- APNG support has been added via the APNG patch:
  http://sourceforge.net/projects/libpng-apng/

To obtain the original libpng-1.6.40, visit:
http://www.libpng.org/pub/png/libpng.html
